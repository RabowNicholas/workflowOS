- CLAUDE.md — locked — always, start of session
- INDEX.md — live — start of session, to see what exists
- ACTIVE.md — live — start of session, for cross-project status and pointers to project loops
- SESSIONS.md — live — start of session, for recent session history
- REFERENCE.md — in-flux — any task involving voice, copy, content, brand, or output quality
